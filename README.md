Quick Notes

Quick Notes is a simple full-stack web application where users can add, view, and delete short notes.

Features

* Add a note
* View all notes
* Delete a note

Tech Stack

* Node.js
* Express.js
* MongoDB
* HTML & JavaScript

How to Run

1. Install dependencies:
npm install
2. Add your MongoDB connection string in a `.env` file:
MONGO_URI=your_connection_string
3. Start the server:
npm start
4. Open in browser:
http://localhost:5000
